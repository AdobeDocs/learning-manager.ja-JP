# 概要

バーチャルコーチは、理論上の知識と実際の応用との決定的なギャップを埋めるように設計された、Adobe Learning Managerの革新的な機能です。 受動的な学習方法、ビデオ、クイズ、読解教材だけに頼るのではなく、 バーチャルコーチは、受講者に臨場感あふれるインタラクティブな環境を提供し、高い評価を受ける会話の練習やソフトスキルの習得を促します。

バーチャルコーチは「24時間週7日練習パートナー」や「コミュニケーションのためのフライトシミュレータ」と考えてください。 パイロットが実際の航空機を操作する前にフライトシミュレータで数千時間を記録するのと同様に、従業員は実際のクライアント、顧客、またはチームメンバーと同じ状況に遭遇する前に、AIを活用したペルソナで複雑な会話を練習できます。 バーチャルコーチは、学習者からの入力にリアルタイムで適応し、学習者が何を言ったか（知識）と言い方（配信とスタイル）の両方に関するデータ駆動型のフィードバックを即座に提供します。
