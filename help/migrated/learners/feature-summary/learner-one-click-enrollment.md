# Adobe Learning Managerでのワンクリック登録

管理者からのディープリンクをメール、チャット、その他のメッセージングプラットフォーム経由で受け取り、トレーニングモジュールまたはコースに登録します。 リンクを選択して即座に登録し、追加の手順なしでモジュールを起動します。

## 学習者を登録し、コースにリダイレクトする

以下のシナリオと対応するアクションに注意してください。 学習者が、自分に送信されたディープリンクをクリックすると、状況に応じて次のいずれかが発生します。

| SN | シナリオ | アクション |
| --- | --- | --- |
| 1 | 単一モジュールのコースおよび順序なし | 自動登録され、プレーヤーが開きます。 ビデオを再生するには、「再生」を押す必要があります。 |
| 2 | マルチモジュールコースと順序なし | 自動登録され、ディープリンクがリンクされている特定のモジュールに移動します。 ビデオを再生するには、「再生」を押す必要があります。 |
| 3 | マルチモジュールのコースと注文あり | 「概要」ページが表示され、前のコースを完了していない場合は、現在のコースを開始する前に前のコースを完了する必要があるというメッセージが表示されます。 この場合、自動登録され、コースの最初のモジュールに移動します。 |
| 4 | コースに前提条件がある | 前提条件が完了している場合は、playerでモジュールを起動できます。 再生を押してビデオを再生します。 完了していない場合は、「必須コースをスキップできません。 このコースを開始する前に、必要なすべてのコースを完了してください。 |
| 5 | キャンセル待ちのコース | 登録されます。 コースの概要ページが表示され、キャンセル待ちリストに追加されます。 コースがキャンセル待ちであるというメッセージも表示されます。 |
| 6 | ディープリンクが生成されたコースに既に登録している場合。 | モジュールに進むと、プレーヤーが起動します。 再生を押してビデオを再生します。 |

## 認証ステータスとディープリンクの動作

上記のシナリオに加えて、認証（ログインおよびログアウト）ステータスに関連するユーザー動作が、さらに2つあります。

* ディープリンクをクリックしてもAdobe Learning Managerにログインしていない場合は、ログインページにリダイレクトされます。 ログインすると、ディープリンクによって特定のモジュールにリダイレクトされます。

* ディープリンクをクリックし、Adobe Learning Managerにログインしている場合は、直接モジュールにリダイレクトされます。

## モバイルとwebの没入型サポート

次のシナリオは、Adobe Learning Managerのモバイルバージョンとwebバージョンの両方に適用されます。

* デスクトップ版のAdobe Learning Managerがなく、ディープリンクを受け取った場合、web版またはモバイル版にログインしていない場合は、リンクを選択すると、ブラウザーのAdobe Learning Managerのログインページにリダイレクトされます。
* デスクトップ版のAdobe Learning Managerがなく、ディープリンクを受け取った場合、既にweb版またはモバイル版にログインしている場合は、リンクを選択するとブラウザーのAdobe Learning Managerモジュールに直接リダイレクトされます。

## ブラウザー制限付き自動再生

ディープリンクをクリックすると、ログインしていない場合はモジュールまたはログインページにリダイレクトされます。 モジュールがビデオの場合、プレーヤーはビデオを自動的には再生しません。 いずれの場合も、ブラウザーはプレーヤーによるビデオの自動再生を禁止します。 ビデオを再生するには、手動で再生を押す必要があります。
