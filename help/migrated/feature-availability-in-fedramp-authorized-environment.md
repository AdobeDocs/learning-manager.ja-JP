# FedRAMP承認環境での機能の可用性

## 概要

FedRAMPのセキュリティおよびコンプライアンス要件の一部として、FedRAMPが承認する環境では現在、Adobe Learning Managerの次の機能はサポートされていません。 これらの制限は、規制されたデータの保護、境界コントロールの維持、および必須の監査基準の維持のために設けられています。 当社は、これらの制約を積極的に追跡し、FedRAMPの姿勢が進化するにつれて、将来的な可用性の変化を伝えていきます。
