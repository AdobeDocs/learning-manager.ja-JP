# 複製したReport Builderテンプレートのカスタマイズ

テンプレートを複製した後、保存してダウンロードする前に、列の追加または削除、フィルターの更新、列名の変更、並べ替えの調整を行うことができます。 この記事では、各カスタマイズ手順\について説明します。

開始する前に、「**テンプレート**」タブからテンプレートを複製します。

## 列の追加と削除

1. 編集ビューで、追加する列を含むデータセットを見つけます。
2. 列名の横の&#x200B;**+**&#x200B;を選択します。 列がレポートキャンバスに表示されます。
3. 同じ列を複数回追加する（たとえば、2つの異なるステータス値をカウントする）。 その列に対して&#x200B;**+**&#x200B;をもう一度選択します。
4. 列を削除するには、カンバスで列を選択し、削除アイコンを選択します。

## 列の並べ替えと名前の変更

1. カンバス内の目的の位置に列をドラッグします。 カンバスでの順序は、ダウンロードしたファイルの列の順序と一致します。
2. 列の名前を変更するには、その別名フィールドを選択し、レポートの列ヘッダーとして表示する名前を入力します。

## フィルターを更新

1. 既存のフィルター値を変更するには、フィルター行を選択し、値を編集して確認します。
2. 新しいフィルターを追加するには、**[フィルターの追加]**&#x200B;を選択します。
3. フィルターを適用するフィールドを選択します。
4. 演算子を選択します。 使用可能な演算子は、フィールドのデータ型によって異なります。
5. 値を入力または選択：
   * 文字列フィールドの場合は、検索文字列を入力して候補から選択します。
   * リストフィールドの場合は、ピッカーから1つ以上の値を選択します。
   * 日付フィールドには、日付を入力するか、過去90日間など、相対的な範囲を選択します。
6. フィルターを削除するには、そのフィルター行の削除アイコンを選択します。

## 並べ替えの更新

1. 並べ替えの基準にする列の並べ替えアイコンを選択します。
2. **昇順**&#x200B;または&#x200B;**降順**&#x200B;を選択します。
3. 他の列を基準にソートする場合は、各列に対して同じ操作を繰り返します。 最初のソート列はプライマリ列で、その他のソートはソートの間に適用されます。

>[!TIP]
>
>常に1つ以上の並べ替えを適用します。 並べ替えを行わないと、同じレポートのダウンロード間で行の順序が異なる場合があります。

## 保存してダウンロード

1. 「保存」を選択して、変更内容を「**レポート**」タブに保存します。
2. **ダウンロード**&#x200B;を選択してレポートを生成します。 ファイルの準備が整うと、アプリ内通知が届きます。
