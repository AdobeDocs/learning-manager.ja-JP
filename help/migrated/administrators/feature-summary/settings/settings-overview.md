# 設定

## 概要

Adobe Learning Managerには管理者の詳細な設定が用意されており、組織固有のニーズに合わせて学習管理システムを設定およびカスタマイズできます。 設定はいくつかの主要なカテゴリに分類され、それぞれ異なる管理機能を提供します。

### 基本設定

* このセクションでは、管理者が国/地域、タイムゾーン、ロケールなどの組織の基本的な詳細を設定できます。
* 学習目標のフィードバックメカニズムと評価システムを設定します。
* 全体的なプラットフォームの動作と使用状況に影響するその他の一般設定。

詳しくは、Adobe Learning Managerの[settings](/help/migrated/administrators/feature-summary/settings/basic-settings.md)を参照してください。

### 統合設定

* Adobe IDやシングルサインオン(SSO)などの認証オプションを設定できます。
* ユーザーのプロビジョニング、コースカタログ、およびレポートデータの同期のために、外部データシステムとの統合を管理します。
* 他のLearning Managerアカウントとの接続を設定および管理して、コンテンツ共有や共同学習の取り組みに活用できます。

詳しくは、 Adobe Learning Managerの設定を参照してください。

### 詳細設定

* 学習コンテンツを整理および分類するためのカスタムラベルを作成および管理して、学習者が関連する資料を見つけやすくします。
* 教室の物理的な場所を設定および管理し、インストラクターによるトレーニングセッションを実施します（容量、機器、スケジュールなどの詳細を含む）。
* コンプライアンスとグループの成功ダッシュボードを構成します。

詳しくは、 Adobe Learning Managerの設定を参照してください。
