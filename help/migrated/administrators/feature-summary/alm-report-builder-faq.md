# Report Builder – よくある質問


**テンプレートとレポートの違いは何ですか？**

テンプレートは、Adobe Learning Managerによって提供される事前定義済みのレポート設定です。 これらは一般的なユースケース用に設計されており、すぐにダウンロードでき、読み取り専用です。 これらのファイルは編集できません。 レポートは、独自に保存された設定です。 テンプレートは、最初から作成するか、テンプレートを複製してコピーを編集することで作成できます。 レポートは[**レポート**]タブに表示されます。テンプレートは[**テンプレート**]タブに表示されます。

**テンプレートを直接編集できますか？**

掲示板で テンプレートは読み取り専用です。 テンプレートをカスタマイズするには、[**複製**]を選択して編集可能なコピーを作成します。 変更内容は、[**レポート**]タブに新しいレポートとして保存され、元のテンプレートには影響しません。

重複した行は、データのフィールドがメインレコードと1対多の関係にある場合に表示されます。 一般的な原因としては、複数のインストラクターによるセッション（セッションごとにインストラクターごとに1行）や、複数の作成者による学習目標などがあります。 これを解決するには、**インストラクター名**&#x200B;や&#x200B;**作成者名**&#x200B;など、複数の値を持つフィールドを列として追加します。

**同じレポートの2回のダウンロードで行の順序が変わるのはなぜですか？**

レポート生成では、分散クエリシステムを使用します。 明示的なソートを使用しない場合、行の順序は、どのクエリノードが最初に結果を返すかによって異なり、実行ごとに異なる場合があります。 ダウンロード間で出力の一貫性を確保するには、レポートに少なくとも1つの並べ替えを適用してください。

**日付がUTCで表示されるのはなぜですか？**

Report Builderは、このリリースのUTCの日付値を返します。 アカウントに設定したタイムゾーンが、今後のリリースで日付フィールドに適用されます。 日付ベースのデータを分析する場合は、アカウントのプライマリタイムゾーンに関連するUTCオフセットを考慮します。

**新しい登録データまたは完了データが表示されるまでにどれくらいの時間がかかりますか？**

Report Builderは、Adobe Learning Managerのデータベースから取得されます。このデータベースには、システムでイベントが発生してから約15分の最大レイテンシーがあります。 登録または完了の記録が完了しただけで、レポートに表示されない場合は、少なくとも15分待ってから再度ダウンロードしてください。

**レポートの行または列に制限はありますか？**

レポートは約100万行に制限されています。 このリリースでは、列数に制限はありません。 レポートに100万行以上が必要な場合は、フィルターを適用して範囲を絞り込みます。

**APIまたは自動処理を介してReport Builderデータを取得できますか？**

Report Builderレポートへの自動APIアクセスは、今後のリリースで予定されています。 現在のリリースでは、レポートはサブスクリプションUIを介して手動でダウンロードされるか、Report Builder機能を介してスケジュールに従って受信されます。
