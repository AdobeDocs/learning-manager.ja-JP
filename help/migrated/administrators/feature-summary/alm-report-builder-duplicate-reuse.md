# Report Builderでレポートを複製して再利用する

レポートを複製すると、その列、別名、グループ化の設定、集約、フィルタ、および並べ替えがすべて新しい編集可能なレポートにコピーされます。 別のカタログ、ユーザーグループ、または期間のレポートのバージョンを作成する場合に使用します。

## レポートの複製

1. 複製するレポートを開きます。
2. **アクション** > **複製**&#x200B;を選択します。 レポートのコピーが編集モードで「[元のレポート名] – コピー」という名前で開きます。 例えば、複数マネージャーにわたる構造化されたパフォーマンスビュー – コピー
3. レポートの新しい名前を入力します。
4. フィルターの更新、列の調整、並べ替えの変更など、必要に応じて変更を加えます。
5. **[レポートの保存]**&#x200B;を選択します。

複製されたレポートは&#x200B;**[レポート]**&#x200B;タブに保存され、元のレポートから独立しています。

## テンプレートの複製

「**テンプレート**」タブからテンプレートを複製することもできます。 テンプレートを複製すると、[**レポート**]タブに新しい編集可能なレポートが作成されます。 元のテンプレートは変更されません。

1. 「**テンプレート**」タブを選択します。
2. コピーするテンプレートを探します。
3. 「**複製**」を選択します。
   ![](assets/image049.png)
4. 名前を入力して調整し、**レポートの保存**&#x200B;を選択します。

## ベストプラクティス

* さまざまなチーム、カタログ、または期間のバリエーションが必要な場合は、十分にテストされたレポートをベースとして複製します。 これは、ゼロから構築するよりも高速で、一貫した構造を保証します。
* コピーの名前をすぐに変更して、元のコピーとの違いを反映します。例えば、「コンプライアンスレポートのコピー」ではなく、「VPによるコンプライアンス – APAC」のように変更します。
* テンプレートを複製した後、他の変更を行う前に、レポートが[**レポート**]タブで識別できるように名前を更新してください。
