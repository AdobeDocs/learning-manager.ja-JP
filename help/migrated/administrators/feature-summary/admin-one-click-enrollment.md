# Adobe Learning Managerでのワンクリック登録の設定

## ワンクリック登録の仕組み

ワンクリック登録は、2つのAdobe Learning Manager(ALM)機能が連携して機能する必要があります。
ディープリンクは、ALM内の特定のコースまたは学習目標への直接URLを提供します。 これらのリンク（モジュールへの直接URL）は、電子メール、イントラネットポータル、またはサードパーティアプリケーションに埋め込むことができます。 学習者がリンクをクリックしてもログインしていない場合は、ALMから認証を求めるメッセージが表示され、学習者がコースに直接移動します。

一般に、学習者が業務の最中にSlackやチームを使用している場合や、出張や会議、営業活動に出席する前に2分間の簡単なリフレッシュトレーニングを受ける必要がある場合に役立ちます。コンテンツにすぐにアクセスしてトレーニングを受けることができます。
登録関連サービスは、ディープリンクによってコースプレーヤーが起動する前に、学習者をコースに自動的に登録します。これにより、手動登録ステップが削除されます。このステップが実行されないと、学習者のエクスペリエンスが中断されてしまいます。

学習者がワンクリック登録リンクを選択すると、ALMはAPIを介してバックグラウンドで登録し、ディープリンクを使用してコースにリダイレクトします。 コースプレーヤーがすぐに開きます。

>[!NOTE]
>
>登録はコースレベルでのみ行われ、上位の学習目標（学習パスまたは資格認定）では行われません。

## モジュールのディープリンクの生成

1. Adobe Learning Managerに管理者としてログインします。
2. 左側のナビゲーションペインで&#x200B;**コース**を選択します。
   ![](assets/one-click-enroll1.png)
3. コースを選択
4. **インスタンス**を選択します。
   ![](assets/one-click-enroll2.png)
5. **Module**セクションは、コピーするモジュールの深いリンクを持つインスタンスを選択します。モジュールの詳細は、インスタンスの下部にある展開セクションに表示されます。
   ![](assets/one-click-enroll3.png)
6. リンクをコピーするモジュールに移動します。
   ![](assets/one-click-enroll4.png)
7. 「**リンクをコピー**」を選択します。 これでディープリンクがコピーされました。 このディープリンクは、コース内の特定のモジュールを開くためのリンクです。

このディープリンクを使用して、学習者に送信できるようになりました。
