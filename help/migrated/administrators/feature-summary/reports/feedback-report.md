# フィードバックレポート

## 概要

学習者が学習目標を完了すると、Adobe Learning Managerのフィードバックレポートにレベル1（学習者のフィードバック）とレベル3（マネージャーのフィードバック）の両方が収集されます。 このレポートでは、学習者とマネージャーの主観的な回答と客観的な回答の両方を体系的に表示します。

このレポートでは、名前、電子メール、フィードバックフォーム名、バージョン、言語などの学習者の詳細が追跡され、管理者が定義した質問への回答がキャプチャされます。 また、各質問に対する学習者の選択内容も記録されます（「強く同意する」、「同意する」、「反対」など）。

## 目的とメリット

* コースの質と全体的な学習の有効性を向上させるための実用的なインサイトを提供します。
* ユーザーからの直接のフィードバックに基づいて、ナビゲーション、ペース、および配信方法を調整します。

## ユースケース

* **コンテンツに関する問題を迅速に特定** ：管理者は、サポートチケットやエスカレーションを待つことなく、低い評価や繰り返される否定的なコメントを見つけて、学習目標を更新できます。
* **トレーニングの有効性の測定**:チームは複数のコースまたはバージョンについて学習者のフィードバックを比較し、どの学習目標が適切に機能しているか、また、どの学習目標をやり直す必要があるかを把握できます。
* **フィードバックフォームを使用した学習者のエンゲージメントの追跡** ：管理者は、回答している学習者の数や質問のスキップの数を確認し、フィードバックフォームを絞り込んで回答の質と完了率を向上させることができます。

## フィードバックレポートのダウンロード方法

1. Adobe Learning Managerに管理者としてログインします。
2. 左側のナビゲーションメニューから&#x200B;**[!UICONTROL レポート]**&#x200B;を選択します。
   ![](assets/select-report.png)
   _管理者のホームページでは、レポートをダウンロードするための[レポート]オプションがハイライト表示されています_

3. レポート内の&#x200B;**[!UICONTROL カスタムレポート]**&#x200B;を選択してから、**[!UICONTROL Excelレポート]**&#x200B;を選択します。
4. **[!UICONTROL フィードバックレポート]**&#x200B;を選択します。
   ![](assets/select-feedback-report.png)
   _カスタムレポートセクションには、学習者およびマネージャーのフィードバックデータにアクセスするためのフィードバックレポートを選択するオプションが表示されています_

5. **[!UICONTROL すべてのトレーニング]**&#x200B;または&#x200B;**[!UICONTROL 選択したトレーニング]**&#x200B;と日付範囲を選択します。 2番目のオプションを選択した場合、最大10個のコースまたは学習パスを追加して、フィードバックレポートを生成することができます。 また、レポートは最大1年間生成できます。
   ![](assets/feedback-report.png)
   _トレーニング範囲を選択し、日付範囲を設定して、ダウンロードする前に翻訳オプションを選択することで、フィードバックレポートを構成します_

6. L1フィードバックを翻訳する言語を選択します。 目的の質問とその回答は、言語バージョンが明示的に定義されると、選択した言語に翻訳されます。 選択した言語で明示的に定義された主観的な質問のみがレポートに表示されます。  主観的な質問に対する回答は、元の回答の言語になります。
7. **[!UICONTROL ダウンロード]**&#x200B;を選択して、レポートをダウンロードします。

## フィードバックレポートに含まれる内容

アカウントレベルのレポートのデフォルトの列は次のとおりです。

| 列 | 説明 |
|----|----|
| フィードバックのタイプ | フィードバックが学習者(L1)からのものか、マネージャー(L3)からのものかを示します |
| ユーザー名 | トレーニングを完了した学習者の名前 |
| ユーザーの電子メール | 学習者の電子メールアドレス |
| トレーニング ID | 各学習目標（コース、資格認定、学習パス）に割り当てられる、システム生成の一意のID |
| トレーニング名 | フィードバックが送信される学習項目の名前 |
| トレーニングインスタンス | トレーニングのインスタンス名（複数インスタンスコースの場合） |
| トレーニングの種類 | トレーニングのタイプ（コース、資格認定、学習パス） |
| フィードバックフォームのタイプ | 使用するフィードバックフォームのタイプ/カテゴリ（ミックス済みコース、セルフペースコース、教室コースなど） |
| フィードバックフォーム名 | フィードバックフォームの名前 |
| フィードバックバージョン | フィードバックフォームのバージョン番号 |
| 現在のマネージャー | 学習者のマネージャーの名前 |
| 現在のマネージャーの電子メール | 学習者のマネージャーの電子メールアドレス |
| 完了日 | 学習者がトレーニングを完了した日付 |
| フィードバックの日付 | 学習者がフィードバックを送信した日付 |
| L1フィードバックの元の言語 | 学習者が最初にL1フィードバックを送信した言語 |

フィードバックフォームに追加された4種類の質問に基づいて、アカウントレベルのレポートに次の列が表示されます。

| 列 | 説明 |
|---|---|
| コースの有効性 | フォームに表示された定義済みのコースの有効性に関する質問 |
| コースの有効性の回答 | コースの有効性の質問に対する学習者の回答 |
| NPS質問1 | 最初のネットプロモータースコアの質問 |
| NPS範囲1 | 使用されている評価尺度（例： 0 ～ 10） |
| NPS回答1 | NPSに関する学習者の評価の質問1 |
| Likert質問1 | 第1リッカート級問題 |
| Likert Answer 1 | Likert質問1への回答 |
| テキスト質問1 | フォーム内の最初の自由形式/テキスト質問 |
| 解答テキスト1 | テキスト質問1に対する学習者の回答 |
| L3 Likertスケール質問1 | 学習者のトレーニング後のパフォーマンスを評価尺度を使用して測定します |
| L3 Likertスケール応答1 | このリッカート規模の質問に対するマネージャーの回答 |
| L3自由形式テキストの質問1 | マネージャー向けのL3フィードバックフォームに追加された自由形式の質問。オプションまたは必須として設定可能 |
| L3フリーテキスト応答1 | 自由形式の質問に対するマネージャーの回答 |

>[!NOTE]
>
>アカウントレベルのレポートには、学習者向けに設定されているアクティブフィールドも含まれます。

学習目標レベルレポートには、次の列が表示されます。

| 列 | 説明 |
|---|---|
| 学習者 | 学習者の名前 |
| 電子メール | 学習者の電子メールアドレス |
| フィードバックフォーム名 | フィードバックフォームの名前 |
| フィードバックバージョン | フィードバックフォームのバージョン番号 |
| 学習者の言語 | 学習者が選択した言語 |
