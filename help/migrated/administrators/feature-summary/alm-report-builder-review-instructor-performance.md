# インストラクターのパフォーマンスとReport Builderの確認

このレポートは、トレーニングマネージャーが最もアクティブなインストラクター、リーチする学習者の数、提供するコースを完了した学習者の数を特定する際に役立ちます。

## インストラクターの作業効率レポートの作成

1. Report Builderを起動して、**レポートの作成**&#x200B;を選択します。
2. _インストラクターの効率性_&#x200B;などの名前を入力します。
3. **インストラクター名**&#x200B;を&#x200B;**モジュールセッション**&#x200B;のデータセットから追加します。
4. **モジュールセッションID**&#x200B;を&#x200B;**モジュールセッション**&#x200B;データセットから追加します。 これを集約してセッション数をカウントします。
5. **モジュールのトランスクリプト**&#x200B;のデータセットから&#x200B;**ステータス**&#x200B;を追加します。 カウントを使用して完了数をカウントします。
6. **インストラクター名**&#x200B;の&#x200B;**グループ化**&#x200B;を選択します。
7. **Count**&#x200B;を&#x200B;**モジュールセッションID**&#x200B;に適用します。 エイリアス&#x200B;_合計セッション数_&#x200B;を入力してください。
8. **Count if**&#x200B;を&#x200B;**Status**&#x200B;に適用し、**完了**&#x200B;を選択します。 エイリアス&#x200B;_Total completions_&#x200B;を入力します。
9. 登録数の合計も表示するには、2回目に&#x200B;**Status**&#x200B;を追加します。**Count if**&#x200B;を&#x200B;**未開始**&#x200B;に適用します。エイリアス&#x200B;_登録数_を入力してください。
   ![](assets/image035.png)
10. **インストラクター名**をフィルターして空にしないでください。
   ![](assets/image036.jpg)
11. **完了数**で並べ替え、パフォーマンスの高いインストラクターから順に並べ替えます。
   ![](assets/image037.png)
12. 「レポートを保存」を選択し、**アクション** / **ダウンロード**&#x200B;を選択してレポートをダウンロードします。

ダウンロードしたレポートには、各インストラクターの合計トレーニングセッション、学習者の完了、未開始の登録を比較することで、インストラクターの効率性が集計され、エンゲージメント、完了パフォーマンス、トレーニングフォローアップの潜在的なニーズの評価に役立ちます。

## ベストプラクティス

* カタログラベルを使用して、インストラクターレポートの範囲を特定の部署、場所、またはプログラム\に設定します。 これは、カタログ名のみでフィルタリングするよりも正確です。
* 過去90日間の&#x200B;**登録日**&#x200B;などの日付フィルターを追加して、すべての時間のデータではなく最近の期間にレポートを範囲指定します。
* インストラクター名ではなく、**完了数の合計**&#x200B;など、わかりやすい指標で並べ替えることで、パフォーマンスの違いをすぐに確認できます。
