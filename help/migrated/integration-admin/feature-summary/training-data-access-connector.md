# Adobe Learning Managerのトレーニングデータアクセスコネクタ

## 概要

**トレーニングデータアクセスコネクタ**&#x200B;を使用すると、ヘッドレスラーニングエクスペリエンスを構築して、スタンドアロンにしたり、**Adobe Experience Manager (AEM)サイト**&#x200B;で構築されたカスタムインターフェイスに統合したりできます。 このコネクターを使用すると、検索およびフィルター機能を使用して、学習者に対して最新のトレーニングコンテンツを取得および表示できます。

>[!IMPORTANT]
>
>- この機能は、Adobe Learning ManagerがAdobe Experience Managerに&#x200B;**アドオン**&#x200B;として販売されている場合にのみ使用できます。
>- このコネクタを介して取得されたコースデータは、24時間ごとに更新されます
>- このコネクタは、ヘッドレスまたはAEMベースのログインなしのエクスペリエンスを構築するためのセルフサービスではありません。 ユースケースに適したアプローチを計画するには、Adobeにお問い合わせください。

## 動作の仕組み

コネクタを有効にすると、Adobe Learning Managerは、コース、学習パス、証明書などのトレーニングメタデータを提供する一連のパブリックAPIを公開します。 これらのAPIを使用して、カスタムブランドのフロントエンドを構築できます。このフロントエンドは、トレーニングコンテンツを表示し、検索およびフィルター機能をサポートします。

## トレーニングデータアクセスコネクタの設定

Adobe Learning Managerをデータストレージと検索システムと連携して、トレーニングメタデータをAEM Sitesやその他のヘッドレスエクスペリエンスに転送できます。

コネクタを設定するには、次の手順を実行します。

1. Adobe Learning Managerに統合管理者としてログインします。
2. **「トレーニングデータアクセス」**&#x200B;タイルにカーソルを合わせ、**「接続」**&#x200B;を選択します。

   ![](assets/training-data-access-connector1.png)
   _[接続]を選択して、トレーニングデータアクセスコネクタを構成します_

3. **接続名**&#x200B;を入力してください。
4. **インターフェイスの種類**&#x200B;を選択してください：

   - **Native Learning Manager** ：標準のログイン体験。デフォルトで使用可能です。
   - **ヘッドレスインターフェイス** ：ログインしていないヘッドレスフロントエンド用のパブリックAPIを公開するプレミアムオプションです。

   ![](assets/training-data-access-connector2.png)
   _トレーニングデータアクセスコネクタの構成に必要な詳細情報を入力してください_

5. **Connect**&#x200B;を選択します。 Adobe Learning Managerは、**ベースURL**&#x200B;と&#x200B;**CDN URL**&#x200B;を自動的に生成します。 カスタムサイトまたはアプリでこれらのURLを使用して、トレーニングデータを取得します。

>[!NOTE]
>
>プレミアムプランのお客様は、標準のお客様と異なるAPI URLを受け取ります。

## トレーニングメタデータの書き出し

トレーニングメタデータを書き出すには：

1. コネクタページで「**トレーニングメタデータの書き出し**」を選択します。
2. **この接続を使用してトレーニングメタデータの書き出しを有効にする**&#x200B;を選択して、トレーニングデータの検索および取得システムへの転送を開始します。
3. **[スケジュールの有効化]**&#x200B;を選択し、開始日、時刻、および間隔を設定します。

   ![](assets/training-data-access-connector3.png)
   _トレーニングメタデータの書き出しをスケジュールする_

4. 「**保存**」を選択します。

   - これにより、すべてのコース、学習パス、および証明書の画像が&#x200B;**CDN**&#x200B;に自動的にアップロードされます。
   - また、関連付けられたメタデータが検索システムに書き出されます。

### オンデマンド書き出し

- **必要に応じてエクスポートを実行する：** **必要に応じてエクスポートを実行するには、オンデマンド**&#x200B;に移動し、**開始日**&#x200B;を設定して、**実行**&#x200B;を選択します。
- **実行ステータスの確認：** **実行ステータス**&#x200B;ページでエクスポートの進行状況と履歴を表示します。

## AEMでのwebサイトの構築と公開

ヘッドレスまたはAEM Sitesベースのwebサイトでトレーニングデータを表示するには：

1. **[AdobeのGitHubリポジトリ](https://github.com/adobe/adobe-learning-manager-reference-site/releases/tag/1.0.0)からAEMパッケージ**&#x200B;をインストールします（前提条件）。
2. **ベースURL**、**CDN URL**、**クライアントID**、**クライアントシークレット**&#x200B;および&#x200B;**管理者更新トークン**&#x200B;を使用して、AEMでコンフィギュレーションを作成します。
3. AEMコンポーネントを使用してサイトを構築します。
4. 学習者向けのサイトをPublishします。
5. セットアップの詳細については、[この記事](https://experienceleague.adobe.com/ja/docs/learning-manager/using/integration/aem-sites/adobe-learning-manager-integration-aem)および[この記事](https://experienceleague.adobe.com/ja/docs/learning-manager/using/integration/aem-sites/integrate-aem-learning-manager)を参照してください。

### 学習者の経験

サイトが公開されたら、次の操作を行います。

- Webサイトには、Adobe Learning Managerから検索システムを通じて取得した&#x200B;**コース**、**学習パス**&#x200B;および&#x200B;**証明書**&#x200B;がすべて表示されます。
- 学習者が&#x200B;**ログインしていない**&#x200B;場合、コースの詳細を参照および表示できます。
- 学習者がクリックしてコース、学習パス、または資格認定に登録すると、登録を完了してトレーニングを開始するために&#x200B;**ログイン**&#x200B;するように求められます。

## ログインなしのエクスペリエンス

ログインなしのエクスペリエンスを使用すると、ログインしていないユーザーに対してリアルタイムのエクスペリエンスを作成できます。 例えば、ログインなしの体験は、マーケティングキャンペーンのランディングページとして機能し、新規登録を促します。

Adobe Learning Managerのログインなしのエクスペリエンスは、**Training Data Access**&#x200B;コネクタを使用して設定できます。 コネクタには、次の機能が用意されています。

- 標準サービス
- プレミアム機能

### 標準サービス

標準では、Adobe Learning Managerのネイティブ版を構築します。 ユーザーは、デモンストレーションのみの、ログインなしのヘッドレスエクスペリエンスを構築できます。 デモンストレーションのヘッドレスエクスペリエンスはスケーラブルではないため、実稼働環境では使用しないでください。

### プレミアム機能

このプレミアム製品を使用すると、ユーザーは&#x200B;**トレーニングデータアクセス**&#x200B;コネクタによって構成されているヘッドレスインターフェイスを構築できます。 これにより、ユーザーはコースや学習パスの詳細（名前、説明、作成者、スキル、期間など）に関するデータをリアルタイムで取得できます。ミックス済み学習シナリオの場合は、リアルタイムの人数制限、空席、キャンセル待ちの制限、キャンセル待ちのカウントも取得できます。 お客様は、これらのAPIを使用して、検索機能とフィルター機能を作成し、ログインしていない学習者向けの完全なコース概要を作成できます。

お客様は、この拡張性の高いログインなしのエクスペリエンスを構築するためにプレミアムプランを購入できます。

>[!NOTE]
>
>プレミアムプランを購入するには、サポートチームまたはCSMにお問い合わせください。

ユーザーがプランを購入すると、CSMチームはユーザーのプレミアムプランを有効にします。 Training Data Accessコネクタを使用すると、前述の機能を使用して、ログインなしのエクスペリエンスを設定できます。
