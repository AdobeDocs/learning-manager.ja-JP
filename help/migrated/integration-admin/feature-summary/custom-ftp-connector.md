# Adobe Learning ManagerのカスタムFTPコネクタ

## 概要

Adobe Learning ManagerのカスタムFTPコネクターを使用すると、Adobe Learning Managerと組織のFTP(SFTP)サーバー間で安全かつ自動的なデータ交換を行うことができます。 この統合により、管理者は外部システムからユーザーデータを読み込み、学習者のトランスクリプトやスキルデータをスケジュールに従って書き出すことができます。 このセットアップにより、データ同期が合理化され、手作業が減り、サードパーティの人事またはレポートシステムとのシームレスな統合がサポートされます。 設定には、ITチームとの調整と、Adobeのカスタマーサクセスマネージャー(CSM)の支援が必要です。

>[!NOTE]
>
>カスタムFTP接続を設定するには、カスタマーサクセスマネージャー(CSM)に連絡してください。 設定プロセスでは、ITチームがIPアドレスをホワイトリストに登録したり、必要なポートを開いたり、必要なアクセス権限を持つフォルダを作成したりするための支援を行う場合があります。

## サポートされる機能

カスタムFTPコネクターは、次のアクションをサポートしています。

### データの読み込み

ユーザー読み込みプロセスは、FTPサーバーから従業員データを自動的に取得し、Adobe Learning Managerに読み込みます。 これは、ユーザーデータを含むCSVファイルを生成する複数のシステムを統合する場合に便利です。

- CSVファイルをFTPサーバー上の指定された&#x200B;**import**&#x200B;フォルダーに配置します。
- Adobe Learning Managerはファイルを検出し、必要に応じてファイルを結合して、定義されたスケジュールに基づいてユーザーデータを読み込みます。

このプロセスを自動化する方法については、[スケジュール](/help/migrated/integration-admin/feature-summary/custom-ftp-connector.md#scheduling-reports)セクションを参照してください。

### 属性のマッピング

統合管理者は、CSVファイルの列をAdobe Learning Managerのグループ化可能な属性にマッピングできます。

- マッピングは1回限りの設定です。
- 以降の読み込みにも同じマッピングが使用されます。
- データ構造が変更された場合は、マッピングを再設定できます。

### データの書き出し

Adobe Learning Managerでは、以下を書き出すことができます。

- ユーザースキル
- 学習者のトランスクリプト

これらのレポートファイルはFTPの書き出しフォルダーに保存され、サードパーティ製システムでレポート作成、分析、その他のダウンストリームプロセス用に使用できます。

### レポートのスケジュール

統合管理者は、次の両方をスケジュールできます。

- ユーザーのインポート
- 学習者のトランスクリプトの書き出し

スケジュールを設定することで、Adobe Learning Manager環境をソースシステムに合わせて常に最新の状態に保つことができます。 必要に応じて、日次同期またはカスタム間隔を設定できます。

## カスタムFTPコネクターの設定

カスタムFTPコネクタを設定するには：

1. Adobe Learning Managerに統合管理者としてログインします。
2. **カスタムFTP**&#x200B;タイルにカーソルを合わせ、**接続**&#x200B;を選択します。

   ![](assets/custom-ftp-connector1.png)
   _[接続]を選択してカスタムFTPコネクタを構成する_

### 認証方法を選択

カスタムFTP接続は、次の2つの認証タイプのいずれかを使用して設定できます。

#### 基本認証アカウント

1. 次の情報を入力します。

   - **FTPドメイン**
   - **FTPユーザー名**
   - **FTPパスワード**

   ![](assets/custom-ftp-connector2.png)
   _構成のFTPドメイン、ユーザー名、およびパスワードを入力してください。_

2. **Connect**&#x200B;を選択します。

#### 証明書の認証

FTPサーバーがSSHキー認証をサポートしている場合：

1. **[SSHキーの生成]**&#x200B;を選択します。

   ![](assets/custom-ftp-connector3.png)
   _[SSHキーの生成]を選択してキーをダウンロードします_

2. 公開鍵がコンピューターにダウンロードされます。
3. このキーをFTPサーバーの許可キーリストに追加します。
4. 次の情報を入力します。

   - **FTPドメイン**
   - **FTPユーザー名**
5. **Connect**&#x200B;を選択します。

>[!NOTE]
>
>カスタムFTP構成では、**SFTP**&#x200B;サーバーのみがサポートされています。

## 接続後の設定

接続が確立されると、次のようになります。

- Adobe Learning Managerは、FTPサーバー上に&#x200B;**import**&#x200B;と&#x200B;**export**&#x200B;のフォルダーを自動的に作成します。
- スケジュールとマッピング設定に基づいて、データの読み込みと書き出しを開始できます。
