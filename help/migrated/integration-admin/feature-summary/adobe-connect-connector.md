# Adobe Learning ManagerのAdobe Connectコネクタ

## 概要

Adobe Learning ManagerはAdobe Connectと統合され、バーチャルクラスルームトレーニングの提供と管理を支援します。 この統合により、セッションのスケジュール、永続的または動的な会議室の使用、出席のキャプチャ、クイズ結果の読み込み、学習者へのセッション記録の提供を行うことができます。

## Adobe Connect

Adobe Connectを設定するには：

1. Adobe Learning Managerに統合管理者としてログインします。
2. **Adobe Connect**&#x200B;タイルにカーソルを合わせ、**Connect**&#x200B;を選択します。

   ![](assets/adobe-connect-connector1.png)
   _[接続]を選択して、Adobe Connectコネクタを構成します_

3. 次の情報を入力します。

   - **接続名**
   - **Adobe Connect URL**
   - **管理者の電子メールを接続する**
4. **Connect Admin Login ID**&#x200B;を入力します（Adobe Connectの電子メールログインを使用しない場合に必要）。

   ![](assets/adobe-connect-connector2.png)
   _Adobe Connectの構成に必要な詳細情報を入力してください_

5. **接続認証を有効にする**&#x200B;を選択します。

   >[!NOTE]
   >
   >AdobeホストのConnectアカウントのみがサポートされています。 （ドメインの末尾は.adobeconnect.comである必要があります）

6. **Connect**&#x200B;を選択します。

管理者の電子メールIDを認証したら、次の手順を実行します。

- Adobe Learning Managerに、Connectが統合されていることを示す成功メッセージが表示されます。
- リクエストは、Adobe Connectバックエンドチームに送信されて承認を受けます。 これには通常1 ～ 2日かかります。

>[!IMPORTANT]
>
>Adobe Connectアカウント管理者は、初回ログイン時に利用条件に同意する必要があります。 これを行わないと、ログイン認証に失敗する可能性があります。

## バーチャルクラスルームセッション情報の追加

作成者がバーチャルクラスルームコースのセッションの詳細を入力していない場合、管理者はセッションの詳細を追加できます。

1. 管理者してログインします。
2. VCコースを選択します。
3. **インスタンス**&#x200B;を選択し、**セッションの詳細**&#x200B;を選択します。
4. **編集**&#x200B;アイコンを選択して、セッション情報を追加または更新します。

>[!NOTE]
>
>Connectアカウントには、同時ユーザーが仮想クラスルームを実行するのに十分な会議室と容量が必要です。 常設ルームを使用しない限り、Adobe Learning Managerバーチャルクラスルームセッションごとに新しいConnectの会議室が自動的に作成されます。

## 常設の会議室

Adobe Connectは、永続的な会議室をサポートしており、再利用が可能です。

- Connectの既存の常設会議室を使用して、バーチャルクラスルームセッションを作成できます。
- 代わりに、Adobe Learning Managerに各セッションのダイナミックルームを作成させることもできます。

学習者がAdobe Connectを使用してセッションに参加する場合は、Learning Managerを介して安全な認証を使用して会議室に入ります。

セッションが完了すると、学習者はLearning Managerアプリでセッションの記録とパスコードにアクセスできるようになります。

## Adobe Connect からのクイズスコアの読み込み

Adobe Learning Managerでは、Adobe Connectセッションからクイズデータを読み込み、他のレポートワークフローと組み合わせることができます。 これには、クイズスコア、学習者の反応、完了データ（セルフペースモジュールの仕組みなど）が含まれます。

### クイズ読み込みワークフロー

#### Adobe Connect (Host)

- Connectホストはコースを作成し、インタラクティブクイズを含むコンテンツをアップロードします。
- ホストは、バーチャルクラスルーム(VC)トレーニングを設定して、コースをVCにリンクするか、**「コースを共有」**&#x200B;オプションを使用してセッション中にコースを共有します。

#### Adobe Learning Manager（作成者）

- 作成者は、モジュールの種類が&#x200B;**バーチャルクラスルーム**&#x200B;に設定されたコースをAdobe Learning Managerで作成します。
- **会議システム**&#x200B;ドロップダウンで、VCプロバイダーとして&#x200B;**接続**&#x200B;を選択します。
- Connectでホストが作成した&#x200B;**常設会議室**&#x200B;を選択します。
- インストラクターを割り当て、コースを保存およびパブリッシュします。

#### Adobe Learning Manager（学習者）

- 学習者がコースに登録し、Connect VCセッションに参加します。
- Connectホストを使用すると、学習者はセッションにアクセスできます。

### Adobe Connect（主催者および学習者）

- ホストは、セッションのクイズを共有します。
- 学習者はクイズを完了し、セッションを終了します。

### Adobe Learning Manager（同期および管理者）

- セッションが終了すると、Adobe Learning Managerによってクイズデータが自動的に同期されます。
- クイズ読み込みワークフローは、スケジュールされた期間の終了後に開始されます。
- 進捗状況を管理するために、統合管理者はAdobe Connectコネクタの&#x200B;**実行ステータス**&#x200B;を確認できます。
- インポートが完了すると、状態が&#x200B;**完了**&#x200B;に更新されます。

その後、管理者は読み込まれた結果を確認できます。

- **出席とスコア付け：**&#x200B;最終的なクイズのスコアと出席を表示します。
- **L2クイズスコア：**
   - **ユーザー別：**&#x200B;個々のスコアをポイントとパーセンテージで表示します。
   - **質問別：**&#x200B;クイズの結果をレポートチャートに表示します。
